==== Install Rudder Agent on Debian or Ubuntu

include::../10_install_server/snippets/apt_key.txt.snippet[]

Then add Rudder's package repository:

* on Debian Squeeze:
----

sudo tee /etc/apt/sources.list.d/rudder.list <<EOF
deb http://www.rudder-project.org/apt-2.11/ $(lsb_release -cs) main
EOF

----

* on Ubuntu 12.04 and following, or Debian Wheezy and following:

----

sudo apt-add-repository http://www.rudder-project.org/apt-2.11/

----

Update your local package database to retrieve the list of packages available on our repository:

----

sudo aptitude update

----

Install the +rudder-agent+ package:

----

sudo aptitude install rudder-agent

----
